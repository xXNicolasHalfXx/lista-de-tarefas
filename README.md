# Todo List

Aplicação simples de lista de tarefas feita com HTML, CSS e JavaScript puro.

## Funcionalidades
- Adicionar tarefas
- Remover tarefas
- Interface simples e dinâmica
- Não precisa recarregar a página

## Como executar
1. Baixe o projeto
2. Abra o arquivo index.html no navegador

## Tecnologias
- HTML
- CSS
- JavaScript (Vanilla JS)
